## How to Add Screenshots

1. Take screenshots using your operating system's screenshot tool
2. Save them in this directory with the recommended naming convention
3. Optimize images to reduce file size (use tools like ImageOptim, TinyPNG, etc.)
4. Update the main README.md to reference these screenshots

## Image Optimization

Before committing screenshots:
- Compress PNG files to reduce size
- Use PNG format for UI screenshots (better quality)
- Maximum recommended file size: 500KB per image
- Recommended resolution: 1920x1080 or lower
